# Security Policy

## Reporting a Vulnerability

Please report vulnerabilities privately via GitHub Security Advisories:

- https://github.com/tannernicol/agent-conclave/security/advisories/new

Please include:

- Affected version or commit
- Reproduction steps
- Impact assessment
- Suggested remediation (if known)

## Scope

In scope:

- Code and scripts in this repository
- Default configuration and documented workflows
- Outputs generated by documented commands

Out of scope:

- Misconfigurations in downstream/private deployments
- Vulnerabilities in external dependencies unless exploitable through this codebase
- Third-party service outages or provider-side incidents

## Disclosure Timeline

- Acknowledgement target: within 7 days
- Triage target: within 14 days
- Remediation timeline: shared after triage based on severity

## Public Hygiene

Do not post real credentials, private hostnames, private IPs, or personal data in issues, PRs, logs, screenshots, or reports.
